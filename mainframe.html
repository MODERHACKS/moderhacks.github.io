TSJCSS




A superfast css library with tones of features to make your website unique. It contains various classes that helps you to style your webpage. We know and respect the freedom of styling so we designed this gallery in such a way that it contains almost every essential that you would require to style with your choise. Something Missing ? TSJJS




A superfast javascript library with unique structure. It uses what we call the Fibre DOM that is extremely flexible and superlight. It can clone either one or all elements with a specific class / id or other. It can create elements, provide attributes to them as well as style in pure javascript. It can write in CSS as well as HTML, but too can identify their structures. It's a ES5 standard and is supported by IE9+. It can append, prepend, format etc. a content to any element. It can respond to any event and handle it easily, switch, toogle classes as well as add and remove them. We will make it further better in future.

ColoriZe JS




A superfast mini javascript syntax highlighting library but with a unique feature. You can customize the styles of each and every element highlighted in your own way by using what we call Calling Classes . It supports syntax of various languages and will support more in future as we will make it further better.

TSJMaths JS




A superfast minified Maths library which can do many things. It can do operations ( +, -, *, /, % ), find factorial ( n! ), combination, permuntation, co-ordinate geometry, statistics, probability, and more. We will be making it further better in future and add more math terms to it. *NOTE : Some features may not be supported by IE.
