ColoriZe JS


Know About



Styling or highlighting code with CSS looks so awesome. But how much will you style using CSS ? ColoriZe JS is very useful when it comes to style elements of HTML, CSS, Javascript etc. With the use of this you can style your elements with Javascript by just following some steps only !! No need to add a CSS to a single always while you want to make a thing different beacuse ColoriZe JS supports Custom Styling Classes. With the help of these classes you can style each and every element either solo or all at a time if you don't like the defauft colors and styles. You gonna learn all the custom classes later in this tutorial. Here is an example in which we used ColoriZe JS to style, especially to highlight the elements.



Example

<!DOCTYPE html>
<html lang="en-IN">
<head>
<style>
#tsj { selector : property !important; }
</style>
<body>
<div id="tsj"></div>
<script>
var a = 2;
document.getElementById("tsj").innerHTML = a;
</script>
</body>
</html>

How To Use



To use ColoriZe JS you have to follow just 2 - 3 steps. You have to use what we call Calling Classes. This classes starts with ColoriZe- with the name of language after it. Here is a chart which shows the specific class for each language.


Languages Calling Class
HTML ColoriZe-html
CSS ColoriZe-css
Javascript ColoriZe-js
PHP ColoriZe-html
My SQL ColoriZe-sql
Java ColoriZe-java
Kotlin ColoriZe-kotlin
Python ColoriZe-python

You noticed that PHP also had the same Calling Class that of HTML. It is beacuse HTML and PHP can be used in the same document yet the HTML Calling Class can determine if it is HTML or PHP and can style the elements efficiently. HTML Calling Class is unique as this can also identify the Calling Classes of both CSS and Javscript inside their specific tags <style> and </style> or <script> and </script> so that you didn't need to write multiple classes for each language in an .html document and by the way you can't too since giving multiple Calling Classes to a single element will crash the code and the styles will not execute. Lets consider "ex" as a programming language. You can use "html" , "css" , "js" etc. in place of "ex". We are using TSJ™CSS in this example to style the div. We used "tsj-code" class of TSJ™CSS to get the div nice. "notranslate" class is used for not allowing translators to translate the div content in any other language. Put the classes as given below in the example :



Calling Class Example

<div class="tsj-code notranslate ColoriZe-ex">
...
</div>

Important One



We designed ColoriZe JS in such a way that you can customise the UI according to your choise but maintaining a thing. You should always use a !important after the value to execute without any errors. You should write something like this example in which we are attempting to change the text color : 

.customtag { color : #008eff !important; }

If you don't write your styles with !important at the end you will see no changes in the UI. Another important thing is to add the javascript function ColoriZe(); inside your <script> and </script> tags. Without it the elements will not get their colors resulting in full black and white finish along with no styles that are installed from first. Here is how to set it up :



JS Function

<script>
ColoriZe();
</script>
